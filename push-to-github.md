# Push to GitHub Instructions

## Step 1: Create Repository on GitHub
1. Go to https://github.com
2. Click "+" → "New repository"
3. Repository name: `foodie-express-app`
4. Make it Public
5. Don't initialize with README
6. Click "Create repository"

## Step 2: Run These Commands
After creating the repository, replace `YOUR_USERNAME` with your GitHub username and run:

```bash
# Add remote origin (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/foodie-express-app.git

# Push to GitHub
git push -u origin main
```

## Step 3: Verify
Your repository should now be live at:
`https://github.com/YOUR_USERNAME/foodie-express-app`
